Paypal Product Sales

Sale settings

Sale Name
Enter an optional name for this sale, e.g. "Father's Day 10% off"
Apply Sale To...
Select whether to apply this sale to a single product or to an entire category.
Product Name/SKU
Select the product to which this sale will apply. The product must exist before you can apply sales to it.
Category
Select the category where the sale should be applied. Category sales are applied to all sub-categories unless overridden by a lower-level category. Any active product sale will override all category sales.
Sale Type
Select whether to deduct a fixed amount or a percentage from the base product price. For Category sales you should normally select "Percent"
Sale Amount or Percent
Enter the fixed amount or percent that will be deducted from the base item price. Percentages should be entered as normal values, e.g. "2.5" for a sale of 2.5% (.025).
Starting Date
Enter the starting date when this sale takes effect. Sales are read in ascending order by starting date, so if there are overlapping periods the first matching one will be applied.
Ending Date
Enter the date when this sale expires. If configured, sales will be removed from the database after this date to improve performance.
