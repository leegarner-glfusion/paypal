Category settings

Category Name

Parent Category
Select the parent for this category, or select "--Top--" for a top-level category.
Description
Enter a brief description for this category
Description
Enter the full description for this product. This will be displayed on the product detail screen.
Category Image
Upload an optional image for this category
Enabled
Check to enable this category. Disabled categories are not displayed in the product listing and their products cannot be purchased.
Visible To
Select the user group that is allowed to view and purchase products from this category. Normally this should be "All Users", but in some cases you may wish to restrict purchases of some products to specific members.
