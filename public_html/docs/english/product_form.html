Paypal Product Information

Product settings

Name/SKU
This is the short product name or item number. This will be submitted to Paypal and shown on the customer's receipt.
Category
Select the category for this product.
Short Description
Enter a short description for this product. This will be displayed under the product name in the product lists.
Description
Enter the full description for this product. This will be displayed on the product detail screen.
Keywords
Enter any additional search terms for this product. The name, short description and full description are already searched, so this field should contain words that do not appear in any of the other fields.

These words will not appear anywhere with the product.

Price
Enter the unit price for this product. Only numbers should be entered here- no commas or currency symbols other than a decimal separator..
Images
You may upload one or more images, up to the maximum number and size set in the Paypal configuration. You may also delete existing images by clicking the "Delete" link next to the image thumbnail.

These images will be shown as thumbnails, and can be expanded using the Slimbox viewer. You can also use the advanced editor to incorporate images directly into your product description.

You might want to have at least one image to serve as a product thumbnail.

Enabled?
This checkbox allows you to easily enable or disable products. A product that is disabled will not appear in the catalog or search results.
Taxable?
Select whether this product is subject to sales tax. The default is "true", which means that sales tax will be applied according to your Gateway account configuration. If you have not configured the gateway to automatically apply sales tax, then this checkbox has no effect.

The primary purpose for this checkbox is to make certain items non-taxable.

>Buttons
Select the button types that will be available for this product. To improve performance, especially if encrypted buttons are used, the buttons are created and stored with the product record each time is it updated.

If the shopping cart is enabled, the Add to Cart button is always shown.

Include in Blocks
Select which blocks should include this product. Checking one or more of the checkboxes will cause this product to appear in the associated block, subject to the block's configuration.
Comments
Select the comment option for this specific product. Comments will only be shown at all if the global configuration allows it; this field allows special handling per product.

Yes
Comments will be shown and users can add comments.
No
No comments will be shown.
Closed
Comments will be shown, but no additional comments will be accepted.
Allow Ratings
Check this box to allow site visitors to rate this product.
Text Fields
Enter the prompts for text fields, separated by the "|" character. When the product is viewed these fields will allow user input. This can be used to collect custom text information such as names to engrave or other special instructions.
Product Type
Select the type of product being sold. The current options are
  • Downloadable : A downloadable product such as a piece of software. This type may be priced at zero (free), or may have a price value. It must be associated with a downloadable file.
  • Physical : A physical product that will be shipped to the customer. Physical products must have a non-zero price.
  • Other Virtual: Some other type of product. This product type may or may not have a price associated with it; if it is zero then the buyer will be able to enter their own price into the payment gateway. This product type could be used for donations, for example. This product type may optionally be associated with a downloadable file.
Select File (Download Only)
Select an existing file to be downloaded.
Upload a new file (Download Only)
Select and upload a new file for a downloadable product. In the event that you select an existing file and upload a new one, the new file will take precedence.
Weight (Physical only)
This field is used to determine shipping charges, if you've set up a shipping schedule in your gateway account setup. Enter the weight, using the unit of measure that you use in Paypal. If the weight is set to zero, then no shipping charge will be calculated.
Shipping (Physical only)
Select the type of shipping charge to be applied to the item. The options are:
  • No Shipping: No shipping charge will be added.
  • Use Gateway Profile: Shipping will be calculated by the gateway using the shipping charges set up in your business profile.
  • Fixed Amount: Enter a fixed amount per item to be added to the purchase.
  • Shipping Module: Use the shipping module to base shipping on a number of "units".
Expiration Time (Downloadable, Virtual Only)
Enter the number of days after which a purchase will expire. This is normally used for downloadable products to allow a buyer to download the item only within a certain period.
Track Quantity On Hand (Physical Only)
Check the checkbox to indicate that you wish to track the quantity on hand of an item, and enter the current quantity on hand in the space provided. The quantity on hand can be updated at any time.
Quantity Discounts
If you wish to offer product discounts based on quantity purchased, you can enter up to three minimum purchase amounts and a discount to apply as a percentage. For example, to offer 10% off of a purchase of two or more, and 15% off if 5 or more are purchased, enter "2" for the first minimum purchase amount and "10" for the Discount %; and "5" for the second minimum purchase with "15" for the discount. Discounts are applied to attribute prices, if any, as well as the product base price. Discounts are applied during final checkout. If the item is on sale, quantity discounts are applied after calculating the sale price.
Sale Prices
This is a list of sale prices that have been entered for this product. This is for information only and does not include category-based sale prices.
Available
If this product is to be sold only for a limited time, enter the starting and ending dates here. Click the red "X" to reset the fields and make the product available any time.
