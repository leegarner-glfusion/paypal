Paypal Product Discounts

Discount settings

Apply Discount To...
Select whether to apply this discount to a single product or to an entire category.
Product Name/SKU
Select the product to which this discount will apply. The product must exist before you can apply discounts to it.
Category
Select the category where the discount should be applied. Category discounts are applied to all sub-categories unless overridden by a lower-level category. Any active product discount will override all category discounts.
Discount Type
Select whether to deduct a fixed amount or a percentage from the base product price. For Category discounts you should normally select "Percent"
Discount Amount or Percent
Enter the fixed amount or percent that will be deducted from the base item price. Percentages should be entered as normal values, e.g. "2.5" for a discount of 2.5% (.025).
Starting Date
Enter the starting date when this discount takes effect. Discounts are read in ascending order by starting date, so if there are overlapping periods the first matching one will be applied.
Ending Date
Enter the date when this discount expires. If configured, discounts will be removed from the database after this date to improve performance.
