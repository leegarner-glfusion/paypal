Sandbox Location ID
Enter the location ID to use for testing
Sandbox Application ID
Enter the application ID to use for testing
Sandbox Access Token
Enter the personal access token to use for testing
Producdtion Location ID
Enter the location ID to use in production
Production Application ID
Enter the application ID to use in production
Production Access Token
Enter the personal access token to use in production
IPN Url Override
Enter the URL for customers to return for order processing. The default is {site_url}/paypal/ipn/square.php.
Testing (Sandbox) Mode?
Check this box during testing. The Sandbox credentials and other values will be used instead of Production.
Enabled
Check this box to indicate that this gateway is enabled for use. If not checked, the PayPal gateway will not be used.
Buttons
Select the buttons that are supported by this gateway. Payment for a shopping cart is always supported. This is primarily meant to reduce clutter in blocks and product lists, where you may want only one gateway handling immediate purchases.
Order
Enter a number to represent the priority of this gateway (lower number means higher priority).
